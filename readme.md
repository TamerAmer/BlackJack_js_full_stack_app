Project Brief

Browser Game

MVP
Create a browser game based on Blackjack. Model the game logic and then display it in the browser for a user to interact with.

Include betting functionality, whilst playing a game against an AI dealer.

Extensions
Track a history of players and their scores using mongoDB
Add "Split" functionality
Add "Double down functionality"
Style with CSS
